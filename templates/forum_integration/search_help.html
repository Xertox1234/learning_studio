Search Help & Syntax Guide

Quick Start

Simply type your search terms in the search box and press Enter. The search will look through forum posts, topics, courses, lessons, and exercises.

Example: python loops - finds content containing both "python" and "loops"

Advanced Search Syntax

Syntax Description Example
"exact phrase" Search for exact phrase "machine learning"
+required Term must be present +python programming
-excluded Term must not be present python -java
author:username Content by specific author author:instructor
date:YYYY-MM-DD Content from specific date date:2024-01-15
date:>YYYY-MM-DD Content after specific date date:>2024-01-01

Search Examples

Basic Searches
  • python basics
    Find content about Python basics
  • "for loop"
    Find exact phrase "for loop"
  • +django web development
    Must contain "django", may contain "web" or "development"
Advanced Searches
  • author:instructor machine learning
    ML content by instructor
  • python -java -javascript
    Python content, excluding Java/JavaScript
  • "data structures" date:>2024-01-01

Search Tips

Best Practices
  • Start with simple keywords, then add filters if needed
  • Use quotes for exact phrases like "machine learning"
  • Try different word forms (learn, learning, learned)
  • Use the + operator for essential terms
  • Check spelling and try synonyms
Common Issues
  • Search is case-insensitive (Python = python)
  • Very short words (1-2 letters) may be ignored
  • Special characters in search terms may cause issues
  • Too many filters might return no results

Trust Level Filtering

You can filter results by the author's trust level:

TL0 New User
TL1 Basic User
TL2 Member
TL3 Regular
TL4 Leader

Higher trust level users typically provide higher quality content.
